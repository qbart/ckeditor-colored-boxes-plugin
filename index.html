Ckeditor coloredboxes plugin

Colored boxes in CKEDITOR

View project onGitHub

Welcome to CKEDITOR coloredboxes plugin.

Add plugin to CKEDITOR config.js

config.extraPlugins = 'coloredboxes';

Add 'coloredboxes' icon to toolbar, for example:

{ name: 'paragraph', items : ['coloredboxes'] }

3rd party libs

Coloredboxes plugin depends on Farbtastic Color Picker (read more) I took this one because it's my favourite (but feel free to integrate other color pickers).

In order to make Coloredboxes work you need to load Farbtastic in your head section. 

<link rel="stylesheet" type="text/css" href="farbtastic/farbtastic.css" />
<script type="text/javascript" src="farbtastic/farbtastic.js"></script>

If you have problem with integrating Farbtastic into your site, please visit author page here.
